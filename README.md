# Kwazar Crypto Bot 🌌

Telegram-бот для отслеживания курсов криптовалют, объемов торгов и установки персональных ценовых уведомлений (алертов) в реальном времени. Данные запрашиваются напрямую через Binance API.

## 🚀 Возможности (Features)
- **Курсы валют:** Быстрый просмотр цен популярных криптовалют или любой монеты по запросу.
- **Персональные алерты:** Установка уведомлений на достижение определенной цены (бот работает в фоне и проверяет совпадения каждые 30 секунд).
- **Генерация графиков:** Построение свечных и линейных графиков в чате (Matplotlib).
- **Объемы торгов:** Аналитика и вывод суточного объема торгов (24h) по активам.
- **Утренняя рассылка:** Ежедневная автоматическая сводка с топ-3 самыми быстрорастущими монетами за сутки.
- **Админ-панель:** Статистика по количеству пользователей и активных алертов, а также функция глобальной рассылки сообщений.

## 🛠 Стек технологий
- **Язык:** Python 3.14
- **Фреймворк:** aiogram 3.x
- **База данных:** aiosqlite (асинхронный SQLite)
- **Графики:** Matplotlib
- **Деплой:** Docker & Docker Compose

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/binansmatvei051112-afk/kwazar-crypto-bot.git
   cd kwazar-crypto-bot
   ```
Настройте переменные окружения:

Создайте файл .env в корневой директории и добавьте туда ваш токен бота и ID администратора:

## Фрагмент кода .env
```python
BOT_TOKEN="ВАШ_TELEGRAM_ТОКЕН"
ADMIN_ID=ВАШ_TELEGRAM_ID
```

Запустите проект через Docker:

```bash
docker-compose up --build -d
```
